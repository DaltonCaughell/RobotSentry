RobotSentry
===========

An open source Robot Sentry project powered by OpenCV, and focused on versatility in application and abstraction.

Requirements: 
Windows (For now, limited by OpenCV)

Python 2.7
Pyttsx (Text to Speech Support)
Numpy


INSTALLATION NOTES:
copy the file called cv2.pyd into C:/Python27/lib/site-packeges before attempting to run.
